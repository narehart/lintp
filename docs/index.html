Quick Start

Create a lintp.yml configuration file and start linting your project structure:

lintp:
  custom-matchers:
    kebab-case: 'matches($BASENAME, /^[a-z0-9]+(?:-[a-z0-9]+)*$/)'
  config:
    .js: 'kebab-case'
    .dir: 'kebab-case'
